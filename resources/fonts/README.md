# Project Fonts

This directory stores font files bundled with the application.

Current structure:
- `source-sans-3/`
- `source-code-pro/`
- `jetbrains-mono/`
- `ubuntu-sans/`
- `ibm-plex-sans/`

Downloaded sources:
- `Source Sans 3`: Adobe Source Sans repository
- `Source Code Pro`: Adobe Source Code Pro repository
- `JetBrains Mono`: JetBrains Mono repository
- `Ubuntu Sans`: Canonical Ubuntu Sans repository
- `IBM Plex Sans`: IBM Plex repository

Selection behavior:
- Every bundled font can be selected for either UI or editor in the Setting tab.

Current defaults:
- UI: `Ubuntu Sans`
- Editor: `JetBrains Mono`
